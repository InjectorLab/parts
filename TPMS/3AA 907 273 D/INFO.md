# 3AA 907 273 D

## Title
OEM Volkswagen (VW) Tire Pressure TPMS Control Module 3AA907273D 433MHz

## Part Number
3AA 907 273 D

## Type
TPMS Control Module

## Description
OEM tire pressure monitoring system (TPMS) control module for select Volkswagen models. It receives signals from the vehicle's tire pressure sensors and reports tire pressure status to the car.

## Compatibility
- Volkswagen Passat CC (2009–2012, 2012–2017)
- Volkswagen Passat (2011–2015)
- Volkswagen Tiguan (2012–2018)
