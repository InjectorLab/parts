# 4E0 907 277 B

## Title
OEM Audi Tire Pressure TPMS Control Module 4E0907277B 433MHz

## Part Number
4E0 907 277 B

## Type
TPMS Control Module

## Description
OEM tire pressure monitoring system (TPMS) control module for select Audi models. It receives signals from the vehicle's tire pressure sensors and reports tire pressure status to the car.

## Compatibility
- Audi A8 / S8 quattro (2003–2007)
